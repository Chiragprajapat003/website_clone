# 🌐 Website Clones Collection (HTML & CSS)

A collection of responsive **website clones** built using **HTML and CSS**.  
This project focuses on improving frontend design skills, layout structuring, and responsiveness.

---

## 📖 Overview

This repository contains multiple website clones created to practice real-world UI development.  
Each clone is designed to replicate the layout and styling of popular websites, helping to build a strong foundation in frontend development.

---

## 🚀 Clones Included

### 🛍️ Bata India Clone
- Replicated product-based UI layout  
- Focus on clean design and spacing  

### 💻 cPanel Clone
- Dashboard-style interface  
- Practice with structured layouts  

### 🛒 Meesho Clone
- E-commerce UI with responsive design  
- Focus on grid and flexbox  

### 🦊 Mozilla Clone
- Multi-section landing page  
- Typography and layout practice  

### 💰 Paytm Clone
- Payment app UI design  
- Complex layout structuring  

### 🚕 Rapido Clone
- Landing page design  
- Clean and modern UI  

---

## ✨ Features

- 🎯 Pixel-inspired UI designs  
- 📱 Fully responsive layouts  
- 🎨 Clean and structured CSS  
- ⚡ Lightweight (no frameworks)  
- 🧩 Multiple real-world UI implementations
